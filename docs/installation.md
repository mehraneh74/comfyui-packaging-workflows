# Installation Guide

## Requirements

Before using these workflows, make sure you have:

- ComfyUI installed
- Python 3.10+
- NVIDIA GPU with CUDA support
- Required custom nodes installed

---

## Installing ComfyUI

Download and install ComfyUI from the official repository.

After installation, launch ComfyUI and verify that it runs correctly.

---

## Installing Custom Nodes

Some workflows require additional nodes.

Recommended tools:

- ComfyUI Manager
- ControlNet nodes
- Impact Pack
- Advanced sampling nodes

Install missing nodes through ComfyUI Manager.

---

## Installing Workflows

1. Download this repository.
2. Open ComfyUI.
3. Drag the workflow JSON file into the interface.
4. Connect your models.
5. Load your input packaging image.
6. Run the workflow.

---

## Recommended Workflow

For best results:

- Use high-resolution packaging images.
- Keep original graphics clear.
- Use proper lighting references.
- Adjust sampling settings based on GPU power.
