# ComfyUI Packaging Workflows

Professional ComfyUI workflows for creating realistic, high-quality packaging renders using FLUX Kontext and ComfyUI.

---

## Overview

This repository contains production-ready ComfyUI workflows designed specifically for packaging visualization.

The goal is to help designers generate professional renders while preserving the original package structure, graphics, typography, and proportions.

These workflows are suitable for:

- Product packaging
- Portfolio projects
- Client presentations
- Product advertisements
- Brand mockups

---

## Features

- Professional lighting
- Luxury product photography
- Packaging preservation
- FLUX Kontext optimized
- High-resolution rendering
- Easy customization

---

## Requirements

- ComfyUI
- FLUX Kontext
- Required custom nodes

---

## Installation

Clone this repository:

```bash
git clone https://github.com/mehraneh74/comfyui-packaging-workflows.git
```

Open ComfyUI.

Load one of the workflow JSON files.

Replace the input image with your own packaging design.

Run the workflow.
